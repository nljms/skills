# nljms skills

Personal collection of Claude skills, structured as a Claude Code plugin marketplace.

## Install in Claude Code

Add this marketplace, then install a plugin from it:

```
/plugin marketplace add nljms/skills
/plugin install doc-server@nljms-skills
```

Or browse and install interactively after adding the marketplace:

```
/plugin marketplace add nljms/skills
/plugin
```

## Skills

- [`doc-server`](./skills/doc-server) — serve project plans and docs as browsable HTML on a local port, grouped by project and git worktree.
